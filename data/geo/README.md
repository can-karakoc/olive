# Geographic Data

This directory contains GeoJSON files defining Areas of Interest (AOI) and administrative boundaries.

## Files

### `aegean_aoi.geojson`
Bounding box for the Aegean olive oil production region.

**Provinces covered:**
- İzmir (primary)
- Aydın (primary)
- Balıkesir
- Manisa
- Muğla

**Coordinates:** 26.0°E to 29.5°E, 36.5°N to 40.5°N  
**Purpose:** MVP focus area for Milestone 0-2

### `turkey_provinces.geojson` (To be added)
Administrative boundaries for all Turkish provinces at NUTS-3 level.

**Source options:**
1. GADM (Global Administrative Areas) — https://gadm.org/download_country.html
2. Natural Earth Data — https://www.naturalearthdata.com/
3. Turkish Statistical Institute (TÜİK) official boundaries
4. OpenStreetMap via Overpass API

**Required fields:**
- `name_tr`: Province name in Turkish (e.g., "İzmir")
- `name_en`: Province name in English (e.g., "Izmir")
- `nuts3_code`: NUTS-3 code (e.g., "TR31")
- `region`: Geographic region (Aegean, Marmara, Mediterranean, etc.)
- `geometry`: MultiPolygon boundary

### `olive_mask_v1.geojson` (Generated by pipeline)
Vector representation of olive orchard locations from NDVI thresholding.

**Generated by:** `pipeline/mask.py`  
**Method:** NDVI > 0.4 during growing season, stable across 3+ observations  
**Validation:** Visual comparison with known olive areas from literature/Google Earth

## Coordinate Reference System

All files use **WGS84 (EPSG:4326)** for compatibility with:
- Google Earth Engine
- Leaflet/MapLibre GL JS
- Sentinel-2 imagery
- Standard web mapping
